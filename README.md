# 🚀 Prompt Hub — Личная библиотека промтов

Современное веб-приложение для хранения и быстрого копирования промтов для нейросетей (ChatGPT, Midjourney и др.).

## 🌟 Основные возможности
- **Фильтрация по категориям**: Кодинг, Арт, Тексты.
- **Живой поиск**: Мгновенный поиск по названиям и тексту промтов.
- **Копирование в один клик**: Интеграция с Clipboard API.
- **Система заявок**: Пользователи могут отправлять запросы на новые промты прямо на мой Email (через Formspree API).
- **Responsive Design**: Сайт отлично выглядит на ПК и смартфонах.

## 🛠 Технологии
- **HTML5** (Семантическая верстка)
- **CSS3** (Flexbox, Grid, CSS Variables, Animations)
- **JavaScript ES6+** (DOM Manipulation, Array Methods, Local Filtering)
